Building a Chat Extension
=========================

Let's say your team is building dashboards in Tableau that measure performance over time. You see something worth commenting on and want to document it to share with your team. You could invest time and energy into a disconnected and difficult to maintain word document. Or, you could build a Tableau Extension with |ProductName| and ``trexjacket``.

In this tutorial, we'll create a Tableau extension that allows users to write their thoughts directly into the dashboard. Once we're done we'll have something like this:
